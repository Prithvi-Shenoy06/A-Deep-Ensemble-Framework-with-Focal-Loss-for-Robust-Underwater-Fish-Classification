# 🐟 Underwater Fish Species Classification

This project applies deep learning to classify underwater fish species using a Kaggle dataset of fish images.

## 📂 Project Structure

## 📊 Models Used
- EfficientNet-B0
- Swin-Tiny Transformer
- ConvNeXt-Tiny

## 💡 Features
- Class imbalance handled with weighted loss
- Stratified K-Fold Cross Validation
- Mixed-precision training (AMP)
- Ensemble predictions (Hard & Soft Voting)

## 📁 Dataset
[Kaggle Fish Dataset](https://www.kaggle.com/datasets/markdaniellampa/fish-dataset)
